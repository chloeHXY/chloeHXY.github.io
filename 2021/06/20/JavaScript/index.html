JavaScript

Author: Chloe HXY Date: June 20, 2021  0:00:00 Category: Technology Language

📢：js是一种动态类型语言，无需指定变量的类型，统一使用var关键字声明即可。

数值类型 Number

在 JavaScript 中，不管什么数字类型的全部用 var 声明就可以了，而且 JavaScript 中只有一个数字类型：Number。

数组—本质上是Array对象

js中的数组不是定长的，他用着定长数组的形式 干着变长数组的事

同时js的数组中的元素类型可不唯一

undefined 表示所有没有赋值变量的默认值，而 null 则表示一个变量不再指向任何对象地址。

undefined==null (值相同) undefined===null （false）类型不同

排序 sort()

字符串—本质上是String对象

可以是单引号也可以是双引号

转为字符串–toString()

数字、bool值均可以转换为字符串，使用toString（）方法即可

还可以value +”” 实现转化

转为数值–Number()

Number() 可以把任意值（一般是数字字符串）转换成数值，如果要转换的字符串中有一个不是数值的字符，返回 NaN（not a number)

对于数字字符串转数字，可以 str - 0 实现数值化

判断小数/整数

parseInt（）与number不同，它对于非纯数字的字符串，可以截取前面的数字，返回。但是parseInt（）对于小数，也只去小数点前的部分。

一般筛纯字符串用number(), 然后再用parseInt() / parseFloat() 判断整数还是小数

Math 对象

常用属性：

  • E ：返回常数 e (2.718281828…)。
  • LN2 ：返回 2 的自然对数 (ln 2)。
  • LN10 ：返回 10 的自然对数 (ln 10)。
  • LOG2E ：返回以 2 为底的 e 的对数 (log2e)。
  • LOG10E ：返回以 10 为底的 e 的对数 (log10e)。
  • PI ：返回 π（3.1415926535…)。
  • SQRT1_2 ：返回 1/2 的平方根。
  • SQRT2 ：返回 2 的平方根。

常用方法：

  • abs(x) ：返回 x 的绝对值。
  • round(x) ：返回 x 四舍五入后的值。
  • sqrt(x) ：返回 x 的平方根。
  • ceil(x) ：返回大于等于 x 的最小整数。
  • floor(x) ：返回小于等于 x 的最大整数。
  • sin(x) ：返回 x 的正弦。
  • cos(x) ：返回 x 的余弦。
  • tan(x) ：返回 x 的正切。
  • acos(x) ：返回 x 的反余弦值（余弦值等于 x 的角度），用弧度表示。
  • asin(x) ：返回 x 的反正弦值。
  • atan(x) ：返回 x 的反正切值。
  • exp(x) ：返回 e 的 x 次幂 (e^x)。
  • pow(n, m) ：返回 n 的 m 次幂 (nm)。
  • log(x) ：返回 x 的自然对数 (ln x)。
  • max(a, b) ：返回 a, b 中较大的数。
  • min(a, b) ：返回 a, b 中较小的数。
  • random() ：返回大于 0 小于 1 的一个随机数。

TypeScript

ts是面向对象编程语言，是强类型的，支持静态类型，模块，接口

TypeScript 在开发时就能给出编译错误，而 JavaScript 错误则需要在运行时才能暴露。

ts是js的超集，也就是说，ts包含了js。

元组、枚举、任意值、void 类型和 never 类型是 TypeScript 有别于 JavaScript 的特有类型
